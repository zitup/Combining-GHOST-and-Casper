## Gasper

本仓库是对 [Combining GHOST and Casper](https://arxiv.org/pdf/2003.03052) 论文的中文翻译。

以太坊使用名为 Gasper 的共识机制，结合了 Casper FFG 权益证明与 GHOST 分叉选择规则。
